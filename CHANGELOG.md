### Added

- Initial release of lyTUI
- MP3 playback with Pygame
- Synchronized lyrics display from USLT tags
- Karaoke-style current line highlighting
- Playlist management
- Modern TUI design inspired by btop
- Keyboard shortcuts for navigation

- Modular architecture with separate audio and UI modules
- Textual framework for TUI
- Rich markup for styling
- Mutagen for metadata parsing
